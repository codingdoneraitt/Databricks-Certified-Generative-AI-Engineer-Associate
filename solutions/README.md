# Solutions

This directory contains **complete solution notebooks** for every lab in the course.
Each solution is a full copy of the upgraded notebook with all code, explanations,
expected outputs, exam tips, and working examples.

## How to Use

1. **Try the lab notebook first** — work through it on your own in Databricks
2. **If stuck**, open the matching solution here to see the full working code
3. **Compare your approach** — solutions include expected output blocks so you
   can verify your results match
